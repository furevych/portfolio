In this project, a complex dataset with subcolumns was aggregated and processed using Python (matplotlib, pandas).

Calculations were performed, and new columns were created to provide insights into the balance between incomes and housing costs by state.

A flexible function was developed to enable further data manipulation and visualizations.

The code facilitates the visualization and analysis of housing costs, one of the most significant expenditures.

Review the project functional on Kaggle.com

  • Tools: Python (matplotlib, pandas), Google Spreadsheets
  • Source: U.S. Census Bureau, Original dataset title: "Financial Characteristics for Housing Units With a Mortgage"
  • Year: 2024
  • Format: CSV
